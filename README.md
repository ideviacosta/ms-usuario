Este repositorio contiene un microservicio backend desarrollado en Java y Spring Boot, encargado de la gestión de usuarios y procesos de autenticación y autorización dentro de una arquitectura de microservicios.

El servicio expone APIs REST para el registro, login y administración de usuarios, implementando seguridad mediante JWT, validación de roles y control de acceso. La solución está diseñada bajo arquitectura hexagonal (Ports & Adapters), aplicando principios de Clean Architecture y buenas prácticas de desarrollo para garantizar mantenibilidad, testabilidad y escalabilidad.
